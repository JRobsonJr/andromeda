# andromeda

> Sample styleguide built with theme-ui.

## Install

```bash
yarn add andromeda
```

## Usage

```tsx
import React, { Component } from 'react'

import MyComponent from 'andromeda'

class Example extends Component {
  render() {
    return <MyComponent />
  }
}
```
